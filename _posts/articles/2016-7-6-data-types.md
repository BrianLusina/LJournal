Data Types are of two types: Primitive and Non-Primitive types

# Primitive Data Types

+ int
+ boolean
+ String
+ double
+ char
+ float

# Non-primitve Data Types

+ Integer
+ Boolean
+ String
+ Double
+ Character
+ Float

# Mathematical operators

+ Addition +
+ Subtraction-
+ Division /
+ Modulus %
+ Multiplication *

# Comparison operators

+ Greater than >
+ Less than <
+ Less than or equal to <=
+ Greater than or equal to >=
+ Not equal to !=
+ equal to ==

# Logical operators

+ And &&
+ Or ||
+ Not !
